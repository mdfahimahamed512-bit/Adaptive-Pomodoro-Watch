# Adaptive Pomodoro Watch ⏱️

Adaptive Pomodoro Watch is a TypeScript-based productivity tool that tracks study/work sessions, measures user focus, and dynamically suggests optimal work–break cycles.

## Features
- Start/Pause/End work sessions
- Record focus score per session
- Adaptive optimal work time calculation
- Break tracking
- Minimalistic, distraction-free UI
- Fully client-side TypeScript logic

## Tech Stack
- HTML, CSS, TypeScript

## Deployment
1. Upload project to GitHub
2. Import repository into Vercel
3. Deploy instantly

## Use Case
- Helps students and professionals optimize work–break cycles
- Measures cognitive load per session
- Demonstrates TypeScript-based real-time state handling and adaptive logic
